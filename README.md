# Stock Market Dashboard

A dark-themed stock market dashboard showing major indices and correction levels.

## Features

- **6 Major Indices**: DJI, NDX (Nasdaq 100), SPX (S&P 500), RUT (Russell 2000), VWRA, XNAS
- **Correction Levels**: -5% to -30% from 10-year highs with color-coded status
- **VIX Display**: Color-coded volatility index (green < 15, yellow < 25, orange < 35, red ≥ 35)
- **USD/SGD Exchange Rate**
- **Live Refresh**: Attempts to fetch live data via CORS proxy; falls back to `data.json`
- **Mobile Responsive**: Works on phone and tablet

## Setup

### Local development

```bash
# Start a local server
python3 -m http.server 8080
open http://localhost:8080
```

### Update data

```bash
python3 update.py
```

This fetches live prices from Yahoo Finance and writes `data.json`. Run it before pushing to update the dashboard data.

### Deploy to GitHub Pages

1. Create a repo on GitHub
2. Push the code:
   ```bash
   git init && git add -A
   git commit -m "Initial commit"
   git remote add origin https://github.com/YOUR_USER/stock-dashboard.git
   git branch -M main && git push -u origin main
   ```
3. Go to **Settings → Pages → Source: main branch** → Save
4. Dashboard available at `https://YOUR_USER.github.io/stock-dashboard/`

## Data Sources

- **Live**: Yahoo Finance via CORS proxy (may not always work)
- **Fallback**: `data.json` generated by `update.py`
- **Cache**: localStorage for instant page load
